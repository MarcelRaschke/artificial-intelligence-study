# Data정제, 성능개선

## 1. 머신러닝 개발 및 운영배포 라이프사이클  

#### 1. 분석  
: Big Picture  
: Get data  
: Discover,Visualize the data -> Gain Insights : data를 시각적으로 분석하여 필터링할 대상 확인  
#### 2. 전처리(Pre-Processing)

#### 4. 평가  
:결과평가(성능측정,비교)  
#### 5. 성능개선  
: 부스팅,bagging 알고리즘적용  
: +data정제  
#### 6. serving 환경 구성  
: 운영배포를 위한 환경구성(api 방식등)  
#### 7. 배포 파이프라인 구성  
: 주기적 data학습 및 운영모델 반영을 위한 파이프라인 구성(재학습 주기정의 등)
